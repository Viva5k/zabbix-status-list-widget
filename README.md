# Zabbix Status List Widget

## Опис (UA)
Компактний віджет у вигляді таблиці для моніторингу статусу великої кількості пристроїв. Ідеально підходить для відображення доступності (Ping) та ключових метрик з графіками.

**Особливості:**
- Табличний вигляд: Пристрій | Статус | Значення | Графік.
- Розумні спарклайни: підтримка бінарного режиму для статусів та плавних кривих для чисел.
- Інтерактивність: перегляд точних значень при наведенні курсору на графік.
- Візуальна індикація: підсвітка проблемних рядків (Down).
- Налаштування видимості колонок.

## Description (EN)
A compact table-based widget for monitoring the status of a large number of devices. Ideal for displaying availability (Ping) and key metrics with graphs.

**Features:**
- Table view: Device | Status | Value | Graph.
- Smart sparklines: support for binary mode for statuses and smooth curves for numbers.
- Interactivity: view exact values when hovering over the graph.
- Visual indication: highlighting of problematic rows (Down).
- Column visibility settings.

## Встановлення / Installation
1. Завантажте файли репозиторію.
2. Скопіюйте папку в директорію `ui/modules/` вашого Zabbix.
3. Перейдіть в `Administration -> General -> Modules` та натисніть `Scan`.
4. Увімкніть модуль "Status List Widget".
